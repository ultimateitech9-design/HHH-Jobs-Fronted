# Accounts Dashboard Manual

## 1. Accounts Dashboard Ka Purpose

Accounts dashboard revenue, transactions, invoices, subscriptions, expenses, payouts, refunds, reports, client search, live chat aur payment settings manage karta hai.

Accounts person ka daily kaam:

- Payment collections verify karna.
- Invoices monitor karna.
- Subscription renewal/overdue check karna.
- Expenses record karna.
- Payouts and refunds track karna.
- Billing related customer chats handle karna.
- Payment settings maintain karna.

## 3. Overview

Click `Overview`.

Ye revenue, invoices, collections aur settlements ka summary hai.

Sections:

- Revenue and collections cards
- Collections vs expense pressure chart
- Settlement Channels
- Recent Collections
- Recent Invoice Activity

Use:

- Daily financial snapshot.
- Revenue trend check.
- Recent transaction/invoice activity review.

## 4. Transactions

Click `Transactions`.

Ye all payment collections across job postings, subscriptions, invoices and refunds show karta hai.

Filters:

- Status filter choose karo.
- Search field me transaction ID, reference, company, email type karo.

Steps:

1. Status select karo if needed.
2. Search text type karo.
3. Matching transactions dikhenge.
4. Table me transaction ID, reference, customer, email, amount, status, provider, created date jaise fields check karo.
5. Pagination me `Previous` and `Next` use karo.

Use:

- Payment proof verify karna.
- Failed/pending transaction trace karna.
- Customer billing query answer karna.

## 7. Expenses

Click `Expenses`.

Ye internal spend track karta hai.

Create expense entry:

1. Expense title/name fill karo.
2. Category select karo.
3. Department select karo.
4. Amount fill karo.
5. Status choose karo: pending, approved, rejected.
6. Date/details/note fill karo.
7. Click `Add Expense`.
8. Button `Saving...` dikhega.
9. Expense list me add hoga.

Reset form:

1. Click `Reset`.
2. Draft form clear/default ho jayega.

Expense table search:

- Expense
- Category
- Department
- Note
- Status

Use:

- Approved spend track karna.
- Pending/rejected expenses review karna.
- Cost pressure report karna.

## 11. Client Search

Click `Client Search`.

Accounts me ye billing owners, clients, state and commercial records search karne ke liye hai.

Steps:

1. Client/company/email/phone type karo.
2. Audience filter choose karo if needed.
3. State filter choose karo if needed.
4. Click `Search`.
5. Results me owner, source, status, updated date verify karo.

Use:

- Billing query me client identify karna.
- Duplicate customer confusion solve karna.
- Payment and account ownership verify karna.

## 12. Live Chat

Click `Live Chat`.

Ye accounts-transferred chats ke liye hai.

Use:

- Billing query.
- Invoice query.
- Payout query.
- Refund/payment query.

Steps:

1. Chat select karo.
2. Customer messages read karo.
3. Reply type karo.
4. Click `Send`.
5. Solved case me `Resolve`.
6. Closed case reopen karna ho to `Reopen`.

## 13. Payment Settings

Click `Payment Settings`.

Ye settlement details, payout behavior aur collection methods configure karta hai.

Steps:

1. Payment gateway/settlement fields update karo.
2. Payout behavior select/update karo.
3. Collection methods enable/disable/update karo.
4. Click `Save Settings`.
5. Button `Saving...` dikhega.
6. Settings save hongi.

Important:

- Wrong payment settings collections/payouts affect kar sakti hain.
- Update se pehle finance/admin approval confirm karo.

## 14. Accounts Daily Workflow

1. `Overview` se daily finance snapshot dekho.
2. `Transactions` me failed/pending collections check karo.
3. `Invoices` me due/unpaid invoices review karo.
4. `Subscriptions` me overdue renewals identify karo.
5. `Refunds` and `Payouts` queues check karo.
6. `Expenses` me new spend add/update karo.
7. Billing query aaye to `Client Search` use karo.
8. Transferred chat aaye to `Live Chat` se respond karo.
9. End of day `Reports` review karo.

## 15. Common Confusion

- Transaction nahi mil raha: reference ID, email ya company se search karo.
- Invoice unpaid dikha raha hai: transactions me matching payment verify karo.
- Refund status unclear hai: refunds table me refund ID/account search karo.
- Settings update risky hai: before saving confirm karo.
