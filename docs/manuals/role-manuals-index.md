## Common Platform Rules

Login ke baad sidebar se apna module open karo. Sidebar collapsed ho sakta hai, mouse hover karne par expand hota hai. Mobile view me top menu button se navigation drawer open hota hai.

Top right bell icon notifications ke liye hai. Top right profile/avatar click karne par logout option milta hai for internal roles.

Tables desktop par table format me aur mobile par card format me dikh sakte hain. Search enabled table me type karte hi matching records filter hote hain. Pagination ho to `Previous` and `Next` buttons se pages move karte hain.

Sensitive actions jaise delete, force purge, payment paid mark, permissions save, system settings save aur maintenance mode carefully use karein. Ye actions real platform access, finance, visibility ya user experience affect kar sakte hain.
