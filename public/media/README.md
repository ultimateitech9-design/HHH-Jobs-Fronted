# Public media

`hhh-home-story.webm` and `hhh-home-story.mp4` are optimized derivatives of
Mixkit's "Office meeting" stock clip:

- Source: https://mixkit.co/free-stock-video/office-meeting-261/
- License: https://mixkit.co/license/#videoFree
- Local edit: muted nine-second loop, resized to 960x540, with WebM and MP4 encodes

The home hero loads its image poster first and requests the video only after the
page load event, while respecting reduced-motion and data-saver preferences.
